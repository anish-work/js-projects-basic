///////////    To-do Task List    \\\\\\\\\\\


Hey all What is up?

This is sweet and small Project using the basics of HTML, CSS and JavaScript.

CSS Library Used :
Materialize

JS Library Used :
Jquery

Features :
-User Can Keep Track of the to do tasks.
-User can add a new task.
-User can delete a single task.
-User can delete or clear all the tasks from the list.
-The task are stored in Local browser Storage so if you refresh the page the list is still on the window.
-Deleting a task deletes the task from the local storage.
